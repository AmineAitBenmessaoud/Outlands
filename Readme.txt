Ce dépôt contient un jeu vidéo qui a été créé en utilisant la librairie Python. Pour pouvoir lancer et jouer au jeu, vous devez d'abord installer les dépendances requises qui sont répertoriées dans le fichier "requirements.txt". Une fois que vous avez installé toutes les dépendances, vous pouvez lancer le jeu en exécutant le fichier "game.py" à l'aide de Python. Assurez-vous que Python est correctement installé sur votre système. Une fois le jeu lancé, vous pouvez profiter d'une expérience de jeu divertissante. Amusez-vous bien en explorant le monde virtuel et en relevant les défis du jeu. Bonne chance et que la meilleure stratégie gagne !
